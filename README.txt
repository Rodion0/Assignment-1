Trains

Description: 
This program simulates a trains station in at each juncture one randomly chosen trains donates a car to another. This is achieved by a doubly linked list and a random number generator that has been provided 

Installation: 
Download toSubmit.zip
Extract all files into working directory 

Usage: 
To use: 
make trains
make run 

Known Bugs: 
Value calculation is O(n) so trains with a lot of cars may slow down the program
